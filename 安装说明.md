## 🚀 基本使用

### 查看状态
```powershell
python clash_verge_cli_windows.py status
```

### 查看所有配置文件
```powershell
python clash_verge_cli_windows.py profiles
```

### 查看所有代理节点
```powershell
python clash_verge_cli_windows.py proxies
```

### 切换配置文件
```powershell
python clash_verge_cli_windows.py activate "配置文件名"
```

### 测试代理延迟
```powershell
python clash_verge_cli_windows.py test GLOBAL
```

### 选择代理节点
```powershell
python clash_verge_cli_windows.py select "GLOBAL" "节点名称"
```

### 查看流量统计
```powershell
python clash_verge_cli_windows.py traffic
```

### 控制系统代理
```powershell
# 查看系统代理状态
python clash_verge_cli_windows.py sysproxy

# 开启/关闭系统代理
python clash_verge_cli_windows.py sysproxy-set on
python clash_verge_cli_windows.py sysproxy-set off
python clash_verge_cli_windows.py sysproxy-set toggle
```

### 控制 TUN 模式
```powershell
python clash_verge_cli_windows.py tun on
python clash_verge_cli_windows.py tun off
python clash_verge_cli_windows.py tun toggle
```

## 📝 JSON 输出
