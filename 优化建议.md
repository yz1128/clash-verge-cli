## ⚡ 2. 性能优化

### 问题 1：重复读取配置文件
当前代码在每个命令中都会重复读取 YAML 文件。

**优化方案：添加缓存机制**
```python
from functools import lru_cache
import time

class ClashVergeCLI:
    def __init__(self):
        self._cache_time = {}
        self._cache_ttl = 5  # 缓存 5 秒
    
    @lru_cache(maxsize=10)
    def load_yaml_cached(self, file_path: Path, mtime: float) -> Dict:
        """带缓存的 YAML 加载"""
        if not file_path.exists():
            return {}
        with open(file_path, 'r', encoding='utf-8') as f:
            return yaml.safe_load(f) or {}
    
    def load_yaml(self, file_path: Path) -> Dict:
        """自动使用缓存的 YAML 加载"""
        mtime = file_path.stat().st_mtime if file_path.exists() else 0
        return self.load_yaml_cached(file_path, mtime)
```

### 问题 2：API 调用没有超时处理
```python
def test_proxy_latency(self, proxy_url: str, test_url: str = None) -> Optional[int]:
    # 当前只有 timeout=5，但没有连接超时
    response = requests.get(test_url, proxies=proxies, timeout=5)
```

**优化方案：分离连接超时和读取超时**
```python
def test_proxy_latency(self, proxy_url: str, test_url: str = None) -> Optional[int]:
    try:
        proxies = {'http': proxy_url, 'https': proxy_url}
        start = time.time()
        # 连接超时 3 秒，读取超时 5 秒
        response = requests.get(
            test_url, 
            proxies=proxies, 
            timeout=(3, 5),
            allow_redirects=False
        )
        latency = int((time.time() - start) * 1000)
        return latency if response.status_code == 200 else None
    except Exception as e:
        return None
```

## 🛡️ 3. 错误处理增强

### 当前问题
很多地方使用了裸的 `except:` 语句，会捕获所有异常包括 KeyboardInterrupt。

**优化方案：使用具体的异常类型**
```python
# 不好的做法
try:
    # code
except:
    return None

# 好的做法
try:
    # code
except (requests.RequestException, yaml.YAMLError) as e:
    click.echo(f"[X] Error: {e}", err=True)
    return None
except KeyboardInterrupt:
    click.echo("\n[!] Interrupted by user")
    raise
```

### 建议添加的异常类
```python
class ClashVergeError(Exception):
    """Base exception for Clash Verge CLI"""
    pass

class ConfigNotFoundError(ClashVergeError):
    """Configuration file not found"""
    pass

class ProfileNotFoundError(ClashVergeError):
    """Profile not found"""
    pass

class APIError(ClashVergeError):
    """API request failed"""
    pass
```

### 5.3 添加代理节点测速并行化
当前 `test` 命令是串行测试节点，可以改为并行：

```python
from concurrent.futures import ThreadPoolExecutor, as_completed

def test_proxies_parallel(self, proxies: List[str], max_workers: int = 10):
    """并行测试多个代理"""
    results = {}
    with ThreadPoolExecutor(max_workers=max_workers) as executor:
        future_to_proxy = {
            executor.submit(self.test_proxy_latency, proxy): proxy 
            for proxy in proxies
        }
        for future in as_completed(future_to_proxy):
            proxy = future_to_proxy[future]
            try:
                latency = future.result()
                results[proxy] = latency
            except Exception as e:
                results[proxy] = None
    return results
```

### 5.4 添加自动选择最快节点功能
```python
@cli.command()
@click.argument('group_name')
@click.option('--limit', default=10, help='Test top N proxies')
@click.pass_context
def auto_select(ctx, group_name, limit):
    """自动选择延迟最低的节点"""
    client = ctx.obj['client']
    # 测试延迟
    results = client.test_group_latency(group_name, limit)
    # 选择最快的
    fastest = min(results.items(), key=lambda x: x[1] or float('inf'))
    if fastest[1]:
        client.select_proxy(group_name, fastest[0])
        click.echo(f"[OK] Auto-selected: {fastest[0]} ({fastest[1]}ms)")
    else:
        click.echo("[X] No available proxy found")
```

## 🔐 10. 安全性增强

### 10.1 配置文件权限检查
```python
def check_config_permissions(self):
    """检查配置文件权限是否过于宽松"""
    import stat
    for file in [self.verge_config, self.clash_config]:
        if file.exists():
            mode = file.stat().st_mode
            if mode & stat.S_IROTH or mode & stat.S_IWOTH:
                click.echo(f"[!] Warning: {file} has too permissive permissions")
```
